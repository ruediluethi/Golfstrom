Eigenwerte

Aus dem charakteristischen Polynom der Jacobi-Matrix wissen wir über die Eigenwerte λ1, λ2 folgendes:

−spur(Jf) = 1 + γ + 2|q| + q2 |q| > 0 ⇒ λ1 + λ2 < 0

Det(Jf) = λ1∙λ2 = (1+|q|)(γ + |q|) ( 1 + q2 |q| ( α (1+|q|)2 − β γ (γ+|q|)2 ) )

Wobei dieser Teil gerade  d  dq g(q) = g'(q) entspricht.

Ein Gleichgewichtspunkt qi ist stabil, wenn für die Eigenwerte λ1, λ2 < 0 gilt.
